    class XfOOrth::SymHierarchy

    The SymHierarchy class is used to map a hierarchy of symbols to other symbols that are not going to conflict with existing symbols. This class is fairly tightly linked to the SymMap class as the two work closely together to map symbols for fOOrth.

    • mapped - The mapping of the desired symbol.

    Returns:¶ ↑

    A SymEntry or nil if the symbol is not in the hierarchy.

    Note:¶ ↑

    If multiple symbols share the same mapping, the symbol returned is that of the one defined last.

def unmap(mapped)
  @list.each do |st|
    entry = st.unmap(mapped)
    return entry if entry
  end
end
