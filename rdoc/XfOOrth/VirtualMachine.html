    class XfOOrth::VirtualMachine

    The fOOrth virtual machine implementation class.

    Attributes

    ctrl_stack[R]

    The fOOrth control stack. This is mostly used to hold information relating to control structures during compile and interpretation.

    data_stack[R]

    The fOOrth data stack. This is the primary means used to hold data for processing.

    thread[R]

    The thread that this virtual machine belongs to.
